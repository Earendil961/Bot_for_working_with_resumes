Дизайн-документ проекта
“Бот для отправки и анализа резюме”


Команда:
* Шевцов Лев
* Дильдин Илья
Актуальность:
В цифровую эпоху для работодателей как никогда необходима возможность оперативно хедхантить талантливые кадры, а соискателям важно быть замеченными среди многих тысяч конкурентов на то же место. Наш проект направлен на обе проблемы сразу. В его функционал будет входить частичная стандартизация резюме путем опроса соискателя и его первичный анализ на основе ответов, после которого бот будет оценивать кандидатов и передавать информацию работодателям. Таким образом работодателю будут тратить меньше времени на поиск достойных кандидатов, а соискатели с хорошими характеристиками будут иметь возможность быстрее остальных попасть на очное собеседование.
Техническое описание проекта:
Проект предполагает использование современных методов машинного обучения для создания модели, которая будет анализировать резюме,а сам проект будет реализовываться в виде телеграмм бота. 
Для баз данных мы будем использовать pandas. Seaborn мы будем использовать для визуализации графиков реализации ML. А для телеграмм бота мы будем использовать pytelegrambotapi.
